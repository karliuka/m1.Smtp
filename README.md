# Magento Smtp
Extension enables you to easily use your own custom SMTP server for sending mail.

### Configuration
<img alt="Magento Smtp" src="https://karliuka.github.io/m1/smtp/config.png" style="width:100%"/>

Download the plugin here: https://github.com/karliuka/m1.Smtp/archive/1.1.2.zip
Installation
-------

Download the plugin and copy the 'app' and 'skin' folder to the root of your Magento installation.
